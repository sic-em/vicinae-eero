# Eero

Manage your Eero mesh WiFi network from Vicinae.

## Features

- View and control connected devices
- Manage family profiles and parental controls
- Configure guest network and port forwarding
- Monitor network nodes and reboot as needed
- Set up DHCP reservations

## Installation

Build the extension:

```bash
npm run build
```

The extension will be automatically installed to your Vicinae extensions directory.

## Development

Run in development mode:

```bash
npm run dev
```
